# Slide ReactJs

Slide de carros com ReactJs.

## Tecnologias:

- reactJs
  - react-icons

### Utilizar:

```bash
git clone https://github.com/Feliper-Silva/slide.git
```

### inicializar:

```bash
yarn start
```

OR

```bash
npm start
```
